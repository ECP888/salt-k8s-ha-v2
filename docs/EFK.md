# EFK插件部署----后端还未做存储持久化,后续跟进研究

## 1.创建logging命名空间
```
[root@linux-node1 ~]# kubectl create ns logging
```
## 2.给所有节点打上标签

```
[root@linux-node1 ~]# kubectl label nodes beta.kubernetes.io/fluentd-ds-ready=true --all
```

## 3.部署EFK

```
[root@linux-node1 ~]# kubectl apply -f /srv/addons/EFK/

```
